# V3：工具与特质系统

## 设计目标

引入**工具**（主动获取的持久装备）和**特质**（被动触发的能力解锁），创造组合涌现效应。让玩家在不同局中探索不同的"build"，从"每局做同样的事"变成"这局我试试社交流"。

V3完成后，玩家应该能感受到："这局拿到了XX工具，我要围绕它调整策略"，以及"我这个build和上局完全不同"。

---

## 系统概述

```
工具（Tools）：花钱/精力主动购买，持久生效，改变行动的数值或解锁新行动
特质（Traits）：满足条件自动获得，代表玩家角色的"人设"，提供被动加成

工具 × 特质 × V2行动细化 = 组合涌现
```

**设计约束：**
- 每局最多装备 3 个工具（背包限制，迫使选择）
- 特质没有数量上限，但触发条件互斥（很难全拿）
- 工具和特质的效果必须能和V2的行动细化产生交互

## 工具系统

### 获取方式

新增行动："逛二手市场"（1EP，不花钱，刷新可购买工具列表）

- 每次逛市场随机展示3个工具（从完整工具池中抽取）
- 展示后可选择购买其中任意个（花钱）
- 不买也行，下次逛会刷新列表
- 同一个工具不能重复购买
- 背包满（3个）时可以替换（丢弃旧的换新的）

### 工具列表

#### 生产力工具

| 工具 | 价格 | 效果 | 设计意图 |
|------|------|------|----------|
| LeetCode会员 | $300 + 每周$50 | 刷题行动XP+1（从0-2变成1-3） | 刷题流核心装备，但有持续成本 |
| GitHub Copilot | $200 + 每周$30 | 做开源贡献只需1EP（原2EP） | 降低开源门槛，作品集流友好 |
| 机械键盘 | $400 | 看文档/教程XP+0.5 | 稳定流增强，和疲劳系统互动好 |
| 降噪耳机 | $350 | 学习行动不受负面事件"电脑出问题""突发低烧"影响 | 防守型装备，减少随机性干扰 |
| 二手iPad | $500 | 解锁"通勤学习"：在职时每周自动获得0.5 XP到随机技能 | 在职玩家专属，缓解"上班=停止成长"的问题 |

#### 经济工具

| 工具 | 价格 | 效果 | 设计意图 |
|------|------|------|----------|
| 二手ThinkPad | $500 | 解锁"远程兼职"行动（2EP→$350） | 比普通兼职3EP=$280更高效的赚钱方式 |

| 付费简历模板 | $200 | 所有简历通过率+5%（加算） | 便宜但效果适中，万金油 |
| 知识星球会员 | $150 + 每周$20 | 技术私活收入+$100 | 私活流增强，需要配合技能 |

#### 社交工具

| 工具 | 价格 | 效果 | 设计意图 |
|------|------|------|----------|
| 得体的衬衫 | $300 | 面试vibe率+8% | 面试直接加成 |
| LinkedIn会员 | $200 + 每周$40 | 人脉行动每次+2（原+1），但有持续成本 | 社交流加速器 |
| 共享办公月卡 | $350 + 每周$60 | 每周自动+0.5人脉，且人脉≥5时零工收入+20% | 社交+经济复合效果 |

### 工具间的组合效应

**不是硬编码的combo，而是数值上自然产生的协同：**

- **GitHub Copilot + 做开源**：开源1EP就能涨技能+作品集，配合疲劳系统（文档学累了换开源）形成高效循环
- **LeetCode会员 + 刷题**：刷题变成1-3XP，期望1.6，远超看文档的1XP，但有$50/周持续成本和运气成分
- **二手ThinkPad + 知识星球 + 高技能**：技能5时远程兼职不解锁，但技术私活2EP=$350+$100=$450，配合知识星球极高效
- **咖啡机 + 共享办公月卡**：生活费-$80但月卡+$60/周，净省$20但获得被动人脉
- **二手iPad + 在职**：上班也能成长，允许"先就业再择业"的策略

### 工具的经济压力

工具系统自带平衡机制：
- 一次性购买工具占用现金储备（前期买工具=更快逼近破产线）
- 订阅制工具有持续成本（LeetCode $50/周 = 占生活费10%）
- 3个背包格逼迫取舍（不能什么都要）

## 特质系统

### 获取方式

满足条件自动获得，结算时检查。获得特质时在结算面板高亮显示。

**特质一旦获得就永久生效**，但某些特质有副作用。

### 特质列表

#### 成长型特质

| 特质 | 触发条件 | 效果 | 副作用 |
|------|----------|------|--------|
| 卷王 | 连续3周每周学习≥4次 | 所有学习XP+0.5 | "突发低烧"事件概率翻倍 |
| 速通达人 | 任意一个技能从0升到5，耗时≤4周 | 该技能后续升级XP需求-1（最低1） | 无 |
| 全栈选手 | 系统≥4 且 应用≥4 | 可以同时投系统和应用方向的岗位（原本只能投对应方向） | 无 |

#### 求职型特质

| 特质 | 触发条件 | 效果 | 副作用 |
|------|----------|------|--------|
| 面霸 | 累计通过3次面试vibe关 | 面试vibe基础率+10% | 无 |
| 海王 | 单周投递3份简历×2次（累计） | 海投惩罚减半（3份从×0.65变为×0.82） | 无 |
| 谈判专家 | 拒绝过1个offer后又拿到更高薪的offer | 所有offer薪资+10% | 无 |

#### 生存型特质

| 特质 | 触发条件 | 效果 | 副作用 |
|------|----------|------|--------|
| 细水长流 | 存款连续4周保持>$3,000 | 每周生活费-$50 | 无 |
| 斜杠青年 | 累计做零工/私活≥8次 | 所有零工/私活收入+25% | 无 |
| 背水一战 | 现金低于$500 | 所有行动效果+20%（绝境爆发） | 现金回到$1500以上后消失 |

#### 社交型特质

| 特质 | 触发条件 | 效果 | 副作用 |
|------|----------|------|--------|
| 社牛 | 人脉≥7 | 零工/私活收入+30%，面试vibe+5% | 无 |
| 开源达人 | 作品集≥2 且 人脉≥3 | 做开源贡献时额外+1人脉 | 无 |
| 圈内人 | 通过内推获得面试≥2次 | 内推简历通过率倍率从×1.5升到×2.0 | 无 |

### 特质的互斥性

某些特质的触发条件天然冲突，形成build分化：

- **卷王 vs 斜杠青年**：卷王要每周学4次（只剩3EP做别的），斜杠青年要频繁打工
- **细水长流 vs 背水一战**：一个要存钱，一个在破产边缘
- **面霸 vs 海王**：面霸需要精准投递多面试，海王鼓励广撒网
- **速通达人 vs 全栈选手**：速通要专精一个，全栈要两个都到4

## Build示例

### Build 1："开源社交流"

**核心工具**：GitHub Copilot + 共享办公月卡 + 得体的衬衫
**目标特质**：开源达人 → 社牛

**玩法**：
1. 前2周：买Copilot($200)，学技能到3级解锁基础岗位
2. 第3周：买共享办公月卡，开始做开源（1EP=技能XP+作品集+人脉因为开源达人）
3. 中期：人脉快速到7触发社牛，零工收入+30%缓解经济
4. 后期：靠高作品集+高人脉+内推拿offer，技能不需要特别高

**特点**：技能可能只到5-6级，但作品集3/3 + 人脉10 + 内推 = 简历通过率很高

### Build 2："卷王速通流"

**核心工具**：LeetCode会员 + 机械键盘 + 降噪耳机
**目标特质**：卷王 → 速通达人

**玩法**：
1. 前1周：买机械键盘($400)，全力学习触发卷王进度
2. 第2-4周：靠文档(+0.5机械键盘+0.5卷王=2XP/次) + 刷题(LeetCode加成期望2.1XP)疯狂升级
3. 第4周：触发速通达人，后续升级更快
4. 降噪耳机防止被负面事件打断节奏
5. 中后期：技能8+直接投高级岗

**特点**：前期几乎不赚钱，靠初始$2000+咬牙撑到高技能。高风险高回报。

### Build 3："打工皇帝流"

**核心工具**：二手ThinkPad + 知识星球 + 二手iPad
**目标特质**：斜杠青年 → 细水长流

**玩法**：
1. 前2周：学技能到3，立刻就业（外包测试$780/周）
2. 第3周：买ThinkPad，在职期间用剩余3EP做技术私活
3. iPad让在职也能涨技能（被动0.5XP/周）
4. 存够钱触发细水长流，生活费降低
5. 斜杠青年触发后私活收入+25%
6. 经济完全无压力后辞职，全力冲高级岗

**特点**：极其稳健，几乎不会破产。但成长慢，需要更多周数。

### Build 4："赌狗流"

**核心工具**：LeetCode会员 + 付费简历模板 + 得体的衬衫
**目标特质**：背水一战 → 面霸

**玩法**：
1. 前2周：花完大部分钱买工具，技能学到刚够投中级岗
2. 第3周：现金逼近$500，触发背水一战（所有行动+20%）
3. 在背水一战buff下疯狂投递+面试
4. 面试通过触发面霸，后续面试更容易
5. 要么快速上岸，要么破产Game Over

**特点**：真正的roguelite体验——高方差、高刺激。

## 平衡性考量

### 工具经济平衡

| 工具 | 总投入(10周) | 总收益(10周) | 回报周期 |
|------|-------------|-------------|----------|
| 咖啡机 | $250 | $800(省生活费) | ~3周 |
| LeetCode | $300+$500 | 约多5-8次XP | 看运气 |
| Copilot | $200+$300 | 省约5-10EP | ~4周 |
| ThinkPad | $500 | 远程兼职收入差额 | ~5周 |
| 衬衫 | $300 | vibe+8%难以量化 | 1-2次面试 |

**设计意图**：没有明显"最强"工具，每个工具在特定build中最优。

### 特质获取难度

| 特质 | 最快获取周 | 难度 | 评价 |
|------|-----------|------|------|
| 卷王 | 第4周 | 中（需要连续3周全力学习） | 限制了打工和社交时间 |
| 速通达人 | 第5周 | 高（0到5需要7XP，4周内完成需要额外XP来源） | 需要LeetCode或灵感爆发配合 |
| 斜杠青年 | 第5周 | 低（8次零工=每周2次×4周） | 容易但收益适中 |
| 背水一战 | 第4周 | 被动触发 | 风险极高但爆发力强 |
| 面霸 | 第8周+ | 高（需要3次面试且都过vibe） | 后期特质，锦上添花 |
| 开源达人 | 第5周 | 中（需要作品集和人脉同时发展） | 复合路线的里程碑 |

## 实现优先级

1. **工具数据定义 + 商店面板UI** — 框架搭建
2. **工具效果接入现有计算** — 让工具真正生效
3. **特质数据定义 + 检查逻辑** — 特质自动获得
4. **特质进度追踪 + 详情面板UI** — 让玩家能看到进度
5. **平衡性测试** — 确保没有支配性策略
6. **工具/特质combo测试** — 验证4个build示例都可行
