# 🎮 游戏概念文档：Project "Offer 2026" (工作代号)

## 1. 项目愿景 (Vision)
* **核心目标**：打破 0 收入。通过极简的数值养成与资源博弈，真实模拟程序员在温哥华（及全球裁员潮下）的生存与求职困境。
* **核心卖点**：**“真实得扎心”**。将现实中求职的挫败感转化为游戏内可控的成长感，利用“苦中作乐”的自嘲引发程序员群体的共鸣。
* **设计原则**：数值驱动、低美术需求、高重玩价值。

---

## 2. 核心玩法 (Core Loop)
玩家在有限的时间（如 365 天）内，通过分配每日精力值（Energy），在“生存”与“成长”之间寻找平衡。

* **资源管理**：
    * **精力 (EP)**：每日固定 10 点，用于学习、面试或工作。
    * **现金 (Cash)**：应对温哥华高昂的每日开销（房租、吃饭）。
* **上岗机制**：
    * **失业状态**：每日 10 点精力全自由分配，全力刷题/学英语。
    * **在职状态**：获得稳定现金流，但每日可用精力大幅削减（如仅剩 4 点），模拟“边打工边跳槽”的疲惫。
* **技能树**：
    * **硬技能**：C++ (核心得分)、Python (辅助)、Godot (副业收益)。
    * **软技能**：英语 (全局加成)、面试技巧 (降低随机刁难概率)。

## 3. 数值设计模型 (System Design)

### 3.1 角色背景与多样性 (Diversity)
通过初始属性差异设定难度：

| 背景角色 | 初始资金 | 初始 C++ | 特殊 Buff / Debuff | 难度等级 |
| :--- | :--- | :--- | :--- | :--- |
| **大厂幸存者** | $1,200 | 60 | **负债**: 每日需额外扣除房贷利息 | 地狱 (Hard) |
| **新移民 (PR)** | $5,000 | 30 | **语言门槛**: 英语提升速度较慢 | 普通 (Normal) |
| **女性开发者** | $3,000 | 20 | **社会补偿**: 每月领 $500 补助；职场偏见导致特定岗位门槛 +10% | 挑战 (Challenging) |
| **温哥华土豪** | $20,000 | 5 | **资源优势**: 资金充足，可长期全职闭关 | 简单 (Easy) |

### 3.2 核心公式预设
* **入职判定**：
    `MatchScore = (C++ * 0.6) + (English * 0.3) + (Interview * 0.1)`
    * 若 `MatchScore < JobRequirement`：直接拒信。
    * 若 `MatchScore >= JobRequirement`：`入职概率 P = 50% + (MatchScore - Requirement) * 2%`。
* **生存压力**：
    `DailyDeltaCash = (JobIncome * IsEmployed) - (DailyBaseCost + Rent)`

## 4. 表现形式与美术风格 (Aesthetics)
* **风格定位**：**"The Programmer's IDE"** (程序员的集成开发环境)。
* **视觉方案**：
    * 采用深色模式（VS Code / Monokai 配色）。
    * UI 使用 Godot 原生控件，辅以简单的 ASCII Art 或 Icon。
    * 按钮和交互模仿代码编辑器，增加沉浸感。
* **交互音效**：机械键盘青轴触发声、编译成功的“叮”声、面试失败的系统报警音。

---

## 5. 商业化与“打破0收入”策略
1.  **首发平台 (Itch.io)**：开启 "Pay What You Want" 模式，利用“真实求职经历”吸引开发者群体赞助。
2.  **流量转化**：在游戏内嵌入“温哥华生存/求职实战手册”链接，引导至个人博客或付费内容。
3.  **联网版预演**：未来版本引入“共享岗位池”，玩家需竞争有限的高薪岗位，模拟真实内卷环境。

---

## 6. 一周原型 (MVP) 开发路线图
* **Day 1**: 核心框架搭建。实现 `GameManager` 全局单例与难度初始化。
* **Day 2**: 养成逻辑编写。实现点击按钮消耗精力并提升属性。
* **Day 3**: 岗位生成系统。实现不同等级（生存/专业）岗位的随机刷新。
* **Day 4**: 投递判定与界面。完成“投递-结果弹窗”的反馈循环。
* **Day 5**: 时间推进系统。实现“结束这一天”，扣除房租与重置精力。
* **Day 6**: UI 润色。使用 AI (ComfyUI) 生成背景图，优化按钮样式。
* **Day 7**: 测试与发布。导出 Web(HTML5) 格式，上传至 Itch.io。
