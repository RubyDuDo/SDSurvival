# 找工作模拟游戏 — 设计文档

> 一个关于GAP回归职场、AI冲击、左右不讨好的资源管理游戏。  
> 开发工具：Godot（纯文字原型验证阶段）

---

## 一、核心理念

这不是一个关于"如何成功找到工作"的游戏，而是关于**"在没有正确答案的处境里做选择"**的游戏。

题材来源于真实经历：GAP多年回归职场、AI冲击下技能过时、低端岗位嫌弃经验断层、高端岗位门槛过高、学区压力形成硬截止日期。

**目标玩家**：经历过或正在经历求职困境的人，尤其是海外华人群体。

---

## 二、核心游戏循环

### 基本结构
- 游戏以**周**为单位推进
- 每周玩家拥有固定**行动力点数**（默认7点）
- 每周消耗固定**生活开销**
- 初始资金为0则游戏结束
- 游戏总周期为**一年（52周）**，结束时计算总收入

### 可执行行动

| 行动 | 消耗行动力 | 说明 |
|------|-----------|------|
| 升级技能（复习） | 1点 | 已有技能，升级成本低 |
| 学习新技能方向 | 2-3点 | 开拓新方向，持续投入 |
| 打零工 | 2点 | 维持现金流，但占用面试时间 |
| 投简历 | 1点 | 触发简历筛选判定 |
| 准备面试 | 1点 | 提升本次面试成功率 |
| 关注公司 | 1点 | 解锁隐藏岗位和公司内部信息 |

---

## 三、玩家数值系统

### 核心数值

| 数值 | 说明 |
|------|------|
| 行动力 | 每周可分配点数，入职后略微减少 |
| 资金 | 归零则游戏结束 |
| 心理状态值 | 接受过多不匹配工作会下降，影响行动力上限和面试表现 |
| Gap时长 | 自动累计，每周未投简历/面试则增加，影响简历通过率 |

### 经验数值

| 数值 | 说明 |
|------|------|
| 工作经验 | 总工作年限 |
| 行业经验 | 特定行业的经验值，跨行投简历打折 |
| 大厂经验 | 单独计算，影响大厂简历通过率 |
| 短期跳槽次数 | 3周内离职次数，次数过多影响简历印象 |
| 人际点数 | 入职后自动增长，影响内推概率 |

### 技能系统

- 技能分为：**专业技能、通用技能、面试技巧**
- 初始技能**不为0**（角色有工作经验背景），体现"生锈"状态
- 升级曲线：低等级升级便宜（复习），高等级升级昂贵（新内容）
- 技能可能存在**版本过时**状态（如会Python 2但市场要Python 3），需额外行动力"更新"

## 四、UI反馈设计

### 技能条形图对比
投简历/面试时显示三条对比线：
- 🟦 你的当前技能等级
- 🟥 目标岗位要求
- 🟨 市场平均水平

**作用**：让玩家直观感受到差距，每次升级都能看到条形图变化，产生即时满足感。

### 失败原因显示
每次简历被拒或面试失败，显示具体原因：
> "该岗位要求Python 4级，你目前3级"  
> "面试技巧不足，被其他候选人抢走"  
> "本周该公司岗位已满"

---

## 五、市场系统

- 市场上存在多家公司，各自发布岗位
- 市场有**随机波动**：某类岗位阶段性增多或减少
- 公司状态会变化：扩招、裁员传闻、offer毁约等

### 公司数值

| 数值 | 说明 |
|------|------|
| 行业领域 | 与玩家行业经验联动 |
| 公司规模 | 小厂/中厂/大厂，影响面试轮数和薪资 |
| 薪资水平 | 抠门/标准/大方，通常与门槛成正比 |
| 面试轮数 | 大厂多轮，时间成本高 |
| 隐藏信息 | 关注后解锁：毁约率、面试风格、员工评价 |

---

## 六、公司展示页面

### 卡片信息
每家公司显示：公司名、行业、规模、薪资水平、开放岗位、面试轮数、**你的匹配度**（颜色标注）

匹配度颜色：
- 🟢 基本符合，值得投
- 🟡 有差距，努力一下能投
- 🔴 差距大，现在投是浪费

### 隐藏岗位机制
- 玩家最多**关注3家公司**（有限资源，逼玩家认真选择）
- 只有关注的公司才能看到隐藏岗位（急招、内部转岗、低调高薪）
- 关注需消耗1点行动力
- 曾经入职过的公司自动进入关注列表，且可查看更多内部信息

---

## 七、面试系统

### 两道关卡
1. **简历筛选**：基于技能匹配度、Gap时长、工作经验等
2. **面试**：基于面试技巧、技能深度、随机事件

### 多轮面试（针对大厂）
- 一面：技能门槛判定
- 二面：技术深度
- 三面：HR文化匹配（触发"PUA识别"选择题、gap解释等）
- 每轮之间有**等待期**（2-3天），期间行动力继续消耗
- 等待期内玩家需决定：是否继续投其他公司

---

## 八、核心决策张力（五条设计主线）

### 1. 技能方向模糊
技能树每条方向的市场需求会随时间变化，玩家无法确定哪个方向会火，只能观察市场猜测。押注本身就是游戏性。

### 2. 时间是双向消耗
每过一周未投简历/面试，Gap自动+1，影响简历通过率。**不行动也是在付出代价**，逼玩家不能只闷头升技能。

### 3. 换方向有显性代价
切换技能方向时，面试官会质疑职业规划，需消耗额外行动力准备说法，否则面试成功率下降。

### 4. 低端岗位两难
接受低端岗位 → 稳定现金流，但心理状态值下降  
拒绝低端岗位 → 继续烧钱，但保持职业尊严  
这个取舍没有标准答案。

### 5. 技术路线长期押注
技术岗需要高等级技能，周期长但薪资高。非技术方向门槛低但天花板低。玩家必须在早期押注，中途切换成本极高。

---

## 九、事件卡系统

游戏中途随机或条件触发的两难事件，每张卡片提供2-3个选项，没有明显正确答案。

### 示例事件卡

**「两难时刻」**
> 🎉 小厂offer到手！月薪12k，需3天内答复  
> 同时：大厂二面还在等通知，成功率未知  
> 你的现金还能撑4周

选项：
- 接受小厂 → 稳定现金流，大厂流程终止，简历留下短期记录风险
- 拒绝小厂 → 继续等大厂，本周现金继续消耗
- 拖延答复 → 需消耗面试技巧值，失败则两边都没了

---

**更多事件卡方向**
- 猎头推来高薪岗位，但要求立刻入职，你二面还没结束
- 朋友内推，但方向和现在准备的不一样，要不要临时转向
- 小厂突然毁约，offer没了，你已停投两周
- HR面试中触发PUA选项（你能接受弹性工时吗？）
- 前同事内推机会出现（需要足够的人际点数）

---

## 十、人际系统

- 入职后每周自动增加**人际点数**
- 人际点数积累到一定程度，触发"前同事内推"事件
- 内推可跳过简历筛选，直接进入一面
- 内推失败会损耗人际点数
- 离职后原公司保留人际关系，可继续提供内推机会

---

## 十一、角色构建系统（点数分配）

### 基本规则
- 玩家获得初始**构建点数**
- 购买Buff消耗点数，购买Debuff获得点数
- 通过分配点数自定义起始处境

### Buff示例（消耗点数）
| Buff | 效果 |
|------|------|
| 名校背景 | 简历通过率加成 |
| 大厂经验 | 初始大厂经验值不为零 |
| 存款充裕 | 初始资金增加 |
| 人脉广 | 初始人际点数更高 |
| 配偶有稳定收入 | 每周基础开销降低 |

### Debuff示例（获得点数）
| Debuff | 效果 |
|--------|------|
| 有娃 | 每周消耗额外行动力 |
| 房贷/租金压力 | 每周固定开销增加 |
| Gap年限 | 初始简历通过率下降 |
| 健康问题 | 行动力上限降低 |
| 单亲 | 开销翻倍但行动力不变 |

### 隐藏触发
某些属性会在游戏中触发特殊事件：
- 选了"大厂经验" → 中途可能触发"前公司同事内推"
- 选了"Gap年限" → 面试时HR专门追问gap期间经历

---

## 十二、难度系统

难度通过**初始构建点数**体现，不设独立难度选项：

| 模式 | 初始点数 | 隐喻 |
|------|---------|------|
| 普通模式 | 50点 | 条件一般，还有退路 |
| 困难模式 | 35点 | 捉襟见肘，每个选择都很痛 |
| 现实模式 | 20点 | 这就是生活 |

### 成就系统（驱动多周目）
- 用20点通关 → "这就是生活"
- 全程不接任何小厂offer → "宁缺毋滥"
- 带满Debuff通关 → "现实模式真男人"
- 靠内推拿到第一份offer → "人脉的力量"

---

## 十三、开发建议

### 优先级顺序
1. 先跑通核心资源循环（行动力分配→技能→投简历→结果）
2. 加入条形图UI反馈
3. 加入3-5张事件卡
4. 加入公司展示页面
5. 加入人际系统和隐藏岗位
6. 最后加入角色构建和难度系统

### 数值平衡原则
- 关键数值做成配置文件，不硬编码
- 用相对值而非绝对值（技能等级÷岗位要求=基础成功率）
- 先测试极端路线（全程升技能/第一周狂投/只接小厂），确保没有明显最优解

### 目标发布平台
- 原型验证：itch.io
- 正式版考虑：Steam（需$25上架费，受众更大）
- 参加Game Jam可快速获得真实玩家反馈
